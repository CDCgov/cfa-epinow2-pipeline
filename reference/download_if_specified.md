# Download if specified

Download if specified

## Usage

``` r
download_if_specified(blob_path, blob_storage_container, dir)
```

## Arguments

- blob_path:

  The name of the blob to download

- blob_storage_container:

  The name of the container to download from

- dir:

  The directory to which to write the downloaded file

## Value

The path of the file
